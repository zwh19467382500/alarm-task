# UI Widgets - 组件层

## 职责
定义可复用的UI组件，提供通用的界面元素。

## 主要功能
- 可复用的UI组件
- 组件级别的交互逻辑
- 数据展示和格式化
- 组件状态管理
- 事件回调处理

## 设计原则
- 高度可复用
- 单一职责
- 通过回调与父组件通信
- 无状态优先，有状态按需

## 命名规范
- `{widget_name}_widget.dart` 格式
- 例如：`alarm_card.dart`, `time_picker_widget.dart`

## 组件分类
- **展示组件**: 纯展示，无交互
- **交互组件**: 包含用户交互
- **复合组件**: 组合多个基础组件
